Editable Dropdown

Use [editable]="true" is present, the input can also be entered with typing.

HTML Preview
<p-dropdown [options]="country" formControlName="selectedCity" placeholder="Select Value "
[editable]="true" optionLabel="label"></p-dropdown>

Split Button Dropdown

Use p-splitButton to create split button dropdowns as a single button dropdown.

HTML Preview
 <p-splitButton [model]="splitdropdown" label="Primary"
styleClass="p-button-raised p-button-info mr-2 mb-2"></p-splitButton>

Group Dropdown

Options can be grouped when a nested data structures is provided.

{{ group.label }}
HTML Preview
 <p-dropdown [options]="groupedCities" [(ngModel)]="selectedCity" placeholder="Select a City"
    [group]="true">
    <ng-template let-group pTemplate="group">
        <div class="flex align-items-center">
            <span>group.label </span>
        </div>
    </ng-template>
</p-dropdown>

Filter Dropdown

Dropdown provides built-in filtering that is enabled by adding the filter property.

{{ selectedCountry.name }}
{{ country.name }}
HTML Preview
  <p-dropdown [options]="countries" [(ngModel)]="selectedCountry" optionLabel="name" [filter]="true"
    filterBy="name" [showClear]="true" placeholder="Select a Country">
    <ng-template pTemplate="selectedItem">
        <div class="flex align-items-center gap-2" *ngIf="selectedCountry">
            <div>selectedCountry.name </div>
        </div>
    </ng-template>
    <ng-template let-country pTemplate="item">
        <div class="flex align-items-center gap-2">
            <div>country.name </div>
        </div>
    </ng-template>
</p-dropdown>
