Progress with background color

Use color="var(--tb-primary)" to progress bar class with the below-mentioned color variation to set the background color progress bar.

Progress with label

Add labels to your progress bars by placing text within the progress-bar.

HTML Preview
    <!-- Labels Example --> 
    <div>
    <p-progressBar [value]="25" [style]="{'height': '10px','font-size':'0.6rem'}"></p-progressBar>
    </div>

Indeterminate

For progresses with no value to track, set the mode property to indeterminate.

<!-- Labels Example --> 
<p-progressBar mode="indeterminate" [style]="height: '6px'"></p-progressBar>
