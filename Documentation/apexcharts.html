Apexcharts

  1. Charts
  2. Apexcharts
OverviewOfficial Website

Explore the sample JavaScript charts created to show some of the enticing features packed in ApexCharts.

Add Package
npm install ng-apexcharts
Remove Package
npm uninstall ng-apexcharts or you can remove package by removing specific package from package.json
